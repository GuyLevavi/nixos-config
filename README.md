# nixos-config

NixOS flake config ‚Äî Hyprland desktop with Home Manager and Catppuccin Mocha theming.

## Quick start

```bash
rb       # apply all changes  (nixos-rebuild switch --flake /etc/nixos#nixbox)
update   # bump flake.lock, then rb
```

## Structure

```
flake.nix                    # entry point ‚Äî pins nixpkgs-unstable, home-manager, catppuccin-nix
home.nix                     # user environment ‚Äî programs, dotfiles, theming (shared across machines)
hosts/nixbox/
  configuration.nix          # system config ‚Äî boot, hardware, services, users
  hardware-configuration.nix # generated by nixos-generate-config (do not edit)
config/
  hypr/hyprland.conf         # Hyprland keybinds + autostart (read via builtins.readFile)
  mako/config                # notification layout (colors from catppuccin-nix)
  wofi/                      # launcher config + style
  nvim/                      # LazyVim config (symlinked to ~/.config/nvim)
DECISIONS.md                 # rationale for every architectural choice
headless.md                  # WSL + airgapped deployment strategy
```

## Common commands

| Command | What it does |
|---|---|
| `rb` | `sudo nixos-rebuild switch --flake /etc/nixos#nixbox` |
| `update` | `sudo nix flake update /etc/nixos && rb` |
| `gcold` | `sudo nix-collect-garbage --delete-older-than 14d` |
| `nsh ripgrep fd` | `nix-shell -p ripgrep fd` ‚Äî temporary shell |
| `sudo nixos-rebuild switch --rollback` | Roll back to previous generation |
| `sudo nixos-rebuild switch --flake /etc/nixos#nixbox --show-trace` | Debug failed rebuild |
| `sudo nix-env --list-generations --profile /nix/var/nix/profiles/system` | List generations |

## Notes

- **stateVersion**: `"25.05"` in both `home.nix` and `configuration.nix` ‚Äî do not change after install
- **Catppuccin**: use `programs.<name>.enable = true` (not `home.packages`) or theming won't inject
- **LSPs**: provided by `programs.neovim.extraPackages` ‚Äî Mason's `ensure_installed` is `{}` to avoid FHS-broken downloads; ignore `:Mason` "not installed" cosmetic warnings
- **nix-shell**: bash does not exec nushell inside `nix-shell` (guarded by `IN_NIX_SHELL`)
- **Chrome Catppuccin theme**: must be installed manually from the Web Store (extensions can't be declared)
